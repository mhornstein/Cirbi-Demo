# Cirbi MCP Demo

A **short demo** showing how to build a simple MCP server and connect it to Claude using a minimal Python tool.

It is based on the full hands-on workshop:  
👉 https://github.com/mhornstein/AI_Foundations_Workshop

---

## Purpose

This demo is not meant for deep experimentation or full hands-on learning. Instead, it quickly shows the "bottom line" of MCP with a working end-to-end example.

---

## What you will see

- A simple Python function.
- Turning it into an MCP tool.
- Running it as a server.
- Connecting it to Claude Desktop.
- Using natural language to trigger tool actions.
